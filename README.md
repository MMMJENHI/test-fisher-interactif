# 📊 Test de Fisher Interactif

Une application web légère et interactive pour visualiser et interpréter les résultats d'un **Test Exact de Fisher**. 

## 🚀 Démo en Direct
Cliquez sur le badge ci-dessous pour ouvrir l'application :

[![Voir l'Application](https://img.shields.io/badge/Ouvrir-Application%20Interactive-blue?style=for-the-badge&logo=github)](https://mmmjenhi.github.io/test-fisher-interactif/)

---

## 🛠️ Fonctionnalités
* **Visualisation Dynamique** : Graphiques en barres propulsés par `Chart.js`.
* **Interprétation Automatique** : Sections dédiées à la p-value et à l'Odds Ratio.
* **Interface Responsive** : Design moderne avec `Tailwind CSS`.

## 📖 Comment l'utiliser ?
1. Ouvrez le lien de la démo.
2. Consultez la table de contingence 2x2.
3. Obtenez instantanément l'interprétation statistique du test.

## 🧰 Technologies utilisées
* **HTML5 / JavaScript (ES6+)**
* **Tailwind CSS** (via CDN)
* **Chart.js** (Visualisation de données)
