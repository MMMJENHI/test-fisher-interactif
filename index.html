Le Test de Fisher en R

Cette section introduit le concept fondamental du test. Vous y découvrirez son utilité principale et pourquoi il est parfois préféré à d'autres méthodes statistiques pour évaluer les relations entre différentes catégories de données.

Le test de Fisher en R est une méthode statistique utilisée pour évaluer l'association entre deux variables catégorielles.

💡

Quand l'utiliser ?

Il est particulièrement utile pour les petits échantillons de données où le test du Chi-2 pourrait ne pas être approprié en raison des faibles effectifs théoriques.

R utilise la fonction fisher.test() qui est incluse dans le package de base, donc vous n'avez pas besoin de l'installer séparément.

La Table de Contingence

Ici, nous examinons l'exemple spécifique du rapport : l'association potentielle entre le sexe d'une personne et la possession d'un chat. Observez comment les données sont structurées avant l'analyse.

Imaginons que nous voulions savoir s'il y a une association entre le fait d'avoir un chat 🐈 et le sexe 👨👩 d'une personne.

Exécution dans R

Cette section détaille les étapes exactes de programmation pour reproduire l'analyse. La fonction prend comme argument principal une table de contingence.

1. Création de la matrice
# Création de la matrice de données
donnees_matrice <- matrix(c(4, 6, 8, 2), nrow = 2, byrow = TRUE)

# Ajout des noms de lignes et de colonnes pour une meilleure lisibilité
rownames(donnees_matrice) <- c("Hommes", "Femmes")
colnames(donnees_matrice) <- c("Avec chat", "Sans chat")

# Affichage de la table
print(donnees_matrice)
2. Exécution du test
# Exécution du test de Fisher
test_fisher <- fisher.test(donnees_matrice)

# Affichage des résultats
print(test_fisher)

Interprétation des Résultats

Que signifient les résultats calculés par R ? Cette section décortique les trois métriques cruciales fournies par la fonction pour comprendre la relation entre le sexe et la possession d'un chat.

La p-value

C'est la valeur la plus importante de votre résultat.

Si la p-value est inférieure à votre seuil de signification (généralement 0,05), vous rejetez l'hypothèse nulle (H_0) qui postule qu'il n'y a pas d'association entre les deux variables.

🎯 Conclusion pour notre exemple : Si la p-value est très faible (par exemple, p < 0.05), on peut conclure qu'il y a une association statistiquement significative entre le sexe et le fait d'avoir un chat dans l'échantillon.

Odds ratio (Rapport de cotes)

Cet indicateur mesure la force de l'association.

⚖️ Supérieur à 1

Par exemple, un odds ratio supérieur à 1 suggère que l'événement "avoir un chat" est plus probable pour les femmes que pour les hommes (selon la structure de la matrice).

Intervalle de confiance

C'est une plage de valeurs dans laquelle se trouve probablement le véritable odds ratio de la population entière, pas seulement de notre petit échantillon.

1

Règle d'or : Si l'intervalle de confiance ne contient pas la valeur 1, l'association est considérée comme statistiquement significative.
